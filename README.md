# Digitization_Practice
Python practice code

El código usado es de un exámen hecho en python, dicho examen trata de seleccionar una de las tres opciones exixtentes existentes mediante un menú.

La primera opción trabaja con variables, donde se piden dos números y se calcula la diferencia entre ellos para saber si son números cercanos o lejanos.
La segunda opción trabaja con diccionarios, permitiendo introducir un animal y mostrar su sonido, o añadir un nuevo animal si no existe.
La tercera opción es un juego en el que se genera un número aleatorio entre 1 y 50 y el usuario tiene cinco intentos para adivinarlo.
